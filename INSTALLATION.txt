🚀 快速開始（3 步驟）

1️⃣  設定 AI API（可選）
   
   編輯 .env 檔案，加入你的 OpenAI API Key：
   
   AI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxx
   
   如何取得 API Key：https://platform.openai.com/api-keys
   詳細說明請看：AI_SETUP.md

2️⃣  啟動服務
   
   bash start.sh
   
   腳本會自動：
   - 檢查系統需求（Go, Node.js, FFmpeg）
   - 安裝所有依賴
   - 建立前端
   - 啟動後端服務

3️⃣  開始使用
   
   開啟瀏覽器訪問：http://localhost:8080/
   
   上傳狗狗影片，查看 AI 自動分析結果！
